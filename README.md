## Building the Project
    1. Clone GIT repo
        ```
            git clone https://github.com/bla-ce/north.git
            cd north
        ```
    2. Create Build Directory
        ```
            mkdir build
            cd build 
        ```
    3. Run CMake and Build the project
        ```
            cmake ..
            make
        ```
